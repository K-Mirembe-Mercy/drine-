**Drine** is a safe, legal, and interactive video platform inspired by **TikTok** and **YouTube**. Users can stream tutorials, vlogs, music videos, and more. With features like vertical scrolling, video liking, view counters, categories, and a search function, Drine provides a modern platform for discovering and enjoying videos.

---

## **Features**
1. **Vertical Scrolling Feed**  
   - Browse videos with infinite scrolling, similar to TikTok.
   
2. **Like Button and View Counter**  
   - Users can like videos, and each video has a view counter that increases with each view.

3. **Category Filtering**  
   - Filter videos by categories like **Tutorial**, **Vlog**, and **Music**.

4. **Search Bar**  
   - Quickly search for videos by title.

5. **Responsive Design**  
   - Works seamlessly on mobile, tablet, and desktop.

6. **Embedded YouTube Videos**  
   - Videos are embedded from YouTube and are easy to replace with your own content.
